# Vrunda Lad | Product Manager Portfolio

Welcome to the source code for my professional portfolio website. This project is a custom-built, responsive single-page application (SPA) designed to showcase my experience as an Associate Product Manager and AI Builder.

## 🚀 Overview

The portfolio highlights my expertise in product strategy, user research, and AI-powered solutions. It features a modern, high-fidelity UI with glassmorphism design elements and a built-in Dark/Light theme toggle.

### Key Sections:
- **About Me**: Brief bio and my passion for building user-centric digital products.
- **Projects**: Dedicated deep-dives into my core projects like **SalesSaathi** (B2B SaaS) and **TradePilot AI**.
- **Experience**: My professional journey, including my APM role at SLA (Scoutloop).
- **Skills Architecture**: A structured breakdown of my core product skills, tools ecosystem (Figma, Jira, Supabase, etc.), and technical stack.
- **Credentials**: Certifications like the Generative AI Mastermind and Google Cloud AI Workshop.

## 🛠️ Tech Stack

This project is built using native web technologies for maximum performance and simplicity:
- **HTML5**: Semantic and accessible structure.
- **Vanilla CSS3**: Custom design system using CSS variables, flexbox/grid layouts, and advanced animations (no external frameworks like Tailwind or Bootstrap).
- **Vanilla JavaScript**: Lightweight DOM manipulation for the mobile navigation menu, project modal routing, and theme switching.

## 📱 Features
- Fully responsive design for seamless viewing across mobile, tablet, and desktop devices.
- Dynamic Light / Dark mode toggling.
- High-fidelity aesthetic with smooth transitions and premium SaaS-style layouts.
